# 🏋️‍♂️ Landing Page - Stay Fit

Landing page desenvolvida para uma academia local com foco em captação de alunos e geração de leads via WhatsApp.

---

## 🎯 Objetivo do Projeto

Criar uma presença digital moderna e estratégica para aumentar:
- 📈 Número de alunos
- 📲 Contatos diretos pelo WhatsApp
- 💼 Autoridade da academia no digital

---

## 🧠 Estratégia aplicada

- Estrutura de landing page focada em conversão
- Call-to-actions posicionados estrategicamente
- Design responsivo (mobile-first)
- Destaque para benefícios e prova social

---

## 🛠️ Tecnologias utilizadas

- Next.js
- React
- Tailwind CSS
- Vercel (Deploy)

---

## ⚙️ Funcionalidades

- 📱 Layout responsivo
- 🚀 Carregamento rápido
- 📲 Botão direto para WhatsApp
- 🎯 Seções estratégicas (Hero, Benefícios, CTA, Contato)
